# xH all in one

Static PWA bundle cloned from `xh-all-in-one` and rebranded for xH all in one.

## Run locally

```bash
npm start
```

Then open http://localhost:4173 for local testing, or https://nxh1811.github.io/ after GitHub Pages deploys.

## Verify

```bash
npm run check
```

## Deploy Firebase

```bash
npm run deploy
```
